Written Solution

Chocoholics Anonymous has awarded us a contract to write the ChocAnon data processing software. The software must enable the following functionality: accept payments from customers, change the status of members, look up service codes, update the ChocAnon data center, validate memberships, write records, email member and provider reports, update member records, update the provider directory, run the main accounting procedure, and request the provider directory. Each of these use cases is described in detail via “UseCases.html.” The workload for use case descriptions was well distributed across the group with each member completing either 1-2 use cases as well as another task or 3+ use cases. The use case diagram was delegated to one team member. As a team, all of the use cases were defined by reading through Appendix A: “Term Project: Chocoholics Anonymous” and taking note of the verbs associated with the software. After all use cases were noted and all relationships between them defined, one team member created the use case diagram via UMLetino. The glossary was delegated to one member and was completed by carefully scrubbing through Appendix A: “Term Project: Chocoholics Anonymous” while defining any terms used that are not common nomenclature.
