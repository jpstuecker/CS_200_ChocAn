Written Solution

The ChocAn data data system will process and store user and provider data. The software will computerize member validation by storing all membership data into a database. Operators will access this dictionary to update memberships. Billing info will be stored after a user inputs it into a terminal, and this data will go on to be the ETF data record. A dictionary of providers will be stored. All membership and provider data will be stored at the data center. Hardware needed includes the terminals and servers for the data center. Our software will interact with other software by packaging data. Users will interact with our system through a GUI.
